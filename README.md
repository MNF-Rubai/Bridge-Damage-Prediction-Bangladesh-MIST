# Bridge-Damage-Prediction-Bangladesh-MIST

This application is a Graphical User Interface (GUI) developed for predicting the damage category of bridges in Bangladesh using Machine Learning techniques. The prediction model is based on the **Gradient Boosting Classifier**, which categorizes bridge conditions into three damage classes: **A, B, and C**.

The model was trained using a dataset containing **1,141 bridge records** and **11 input features**.

The Python source code (`.py`) and the trained Machine Learning model file (`.pkl`) have been uploaded to this repository. Due to file size limitations, the compiled executable (`.exe`) file has not been included.

## Instructions for Using the GUI

1. **Location Selection:** Select the bridge location within Bangladesh using the dropdown menu. The **Salinity Level** and **Earthquake Intensity (Earthquake Coefficient, Z)** values will be automatically assigned based on predefined regional values.

2. **Dimensional Inputs:** Enter all dimensional parameters in **meters (m)**.

3. **Categorical Variables:** Select the required categorical variables from the available dropdown menus.

4. **Prediction Options:**

   * Click **"Predict (No PDF)"** to display the predicted damage class directly within the GUI.
   * Click **"Predict & Generate PDF Report"** to generate and download a detailed prediction report in PDF format.
